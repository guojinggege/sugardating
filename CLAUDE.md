# Claude Code 开发规范 — 拾光 Lumina

## 项目定位
华语 / 亚裔创作者的 SFW 发现与订阅平台。门类组织（摄影/视频/艺术服务/AI艺术家/直播/社区），**不做按性别交易的人物目录**，不做成人内容。

## 当前阶段
全而静态的前端框架已就位。你的任务是在此基础上**逐步补功能**，不要推倒重写、不要把现有结构改成其它形态。

## 技术约束
- Next.js 14 App Router + TypeScript，样式走 `app/globals.css`（单色设计 token）。
- 新增图片一律走 `components/Placeholder.tsx` 占位，**禁止引入外部图片 URL、禁止用 AI 生成的人物图**填充；真实素材由运营提供授权后替换。
- 组件分层：Page → Section → Card → UI。单文件不超过 ~300 行，超了就拆。
- 客户端交互才加 `"use client"`，能服务端渲染的保持服务端。

## 设计 token（勿随意改）
- 颜色：`--ink #161618` / `--page #F4F4F5` / `--card #fff` / `--line #E8E8EC` / `--live #E5484D`（仅直播态用红）。
- 主操作=纯黑，整体黑白灰单色，避免再引入彩色。
- 字体：Plus Jakarta Sans + Noto Sans SC。

## 每完成一个模块，输出
- 改动的文件清单 + 简述
- 如涉及数据：类型定义 + mock/migration
- 不要只回一句“已完成”

## 路线（按序补）
数据层 → 搜索/筛选 → 认证 → 发布流 → 订阅/支付。
支付若接入：Stripe（SFW 合规边界写死 + 主动审核），不在产品内开放任何成人内容开关。
